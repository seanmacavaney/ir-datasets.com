"kilt"

KILT is a corpus used for various "knowledge intensive language tasks".

  • Documents: Wikipedia articles
  • Repository
  • Paper
  • Leaderboard

5.9M docs

  2. title: str
  3. text: str
  4. text_pieces: Tuple[str, ...]
